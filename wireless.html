NYU Wireless

Before consulting this guide, make sure to go here for console/TV issues, and here for computer relatied issues first and attempt to use the solutions listed there.

Mac

To solve any issues connecting on your Mac computer, try the following:

  1. Click the wifi logo in the upper right of your screen
  2. Click on "Open Network Preferences"
  3. Odds are the small circle next to the network name is yellow rather than green. This is bad. It probably mentions a self-assigned IP. This is bad too. But rejoice, because it's easy to fix.
  4. Click on Advanced in the bottom right corner of the Network window
  5. Click on the TCP/IP tab
  6. Click on Renew DHCP Lease
  7. Watch as the field that says IPv4 Address goes blank and then generates a new number
  8. Give it 15-30 seconds, and you should be good.

PC (Setting up a router on Windows 7)

  1. Open your Network and Sharing Center
  2. Click "Manage Network Settings"
  3. Right-click on nyu and select "Properties"
  4. Make sure the "Connect automatically when this network is in range" and "Connect even if the network is not broadcasting its name" are checked in the Connection tab
  5. Go to the Security tab
  6. Make sure the Security type is "WPA2-Enterprise", the Encryption type is "AES", and the network authentication method is "Microsoft: Protected EAP"
  7. Check the "Remember my credentials for this connection each time I'm logged on"
  8. Go to Advanced Settings
  9. Check "Specify authentication mode" and set it to "User authentication"
  10. Click on "Save credentials" and type in your NYU username and password
  11. Go to the "Settings" tab
  12. "Validate server certificate" should be checked, as should "Class 3 Public Primary Certification Authority" and the three Verisign things in the "Trusted Root Certification Authorities" box
  13. Authentication Method should be "Secured password (EAP-MSCHAP v2)." Make sure "Automatically use my windows logon name and password is NOT checked.
  14. Check the "Enable Fast Reconnect" box.

Ubuntu

Works with Gutsy, not Intrepid

  1. Go to the nextwork manager
  2. Select Wireless Networks
  3. Left click mouse and select "Connect to other wireless network"
  4. Select "WPAS Enterprise" for a "Wireless Security"
  5. Make sure the options that pop up have these filled in:

Network Name is 'nyu'
EAP Method is PEAP
Key Type: Automatic (or, I believe, AES)
Phase2 Type is MSCHAPv2
Identity is your NYU NetID (eg. js123 -- don't include the @nyu.edu )
Password is your NetID password.

For more up-to-date info on connecting to NYU wireless networks with Ubuntu, check here.

WiiU

Follow the instructions here. You may need to pick up a Wii LAN adaptor (USB to Ethernet for the Wii). Otherwise you'll have to make an Ad-Hoc Network through your laptop or setup a router in your room. See the 3DS section for more information on Ad-Hoc.

3DS

You can't connect directly to the NYU internet, but you can set up an Ad-Hoc connection through your computer that the 3DS is capable of connecting to. Pick your OS below:

  • Windows 8
  • Windows 7
  • Mac
You can also try using connectify.
